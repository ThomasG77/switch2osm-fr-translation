Utilisation d'une solution tout en un

 

Tile Drawer

Tile Drawer est un moteur de rendu libre, facile pour commencer, utilisable immédiatement et un serveur utilisant la chaîne d'outils OSM standard pour Amazon EC2, d'autres serveurs virtuels, et des installations complètes de Ubuntu sur n'importe quel matériel. Choisissez une étendue géographique, sélectionnez un style de carte, et Tile Drawer mettra à jour votre système d'exploitation de base avec un serveur de tuiles qui fonctionne.

(Amazon EC2 est assez souvent utilisé pour l'hébergement de tuiles mais a quelques restrictions: voir les explications, en anglais.)

TileMill et MapBox

TileMill est une application précompilée contenant la moteur de rendu Mapnik, un éditeur de code, et un inspecteur de données. Il est disponible pour Windows, Linux, Mac OS X. Il est conçu par MapBox qui fournit une offre du même nom, MapBox, un service d'hébergement payant pour les tuiles créées avec l'application.

Fournisseurs tierce-partie

De nombreux fournisseurs de service OpenStreetMap seront content de travailler avec vous pour héberger les tuiles cartographiques avec les feuilles de style que vous aurez conçu. Voir la liste des fournisseurs.
