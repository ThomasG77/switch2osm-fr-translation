Reprenez le contrôle sur vos cartes

Es-ce que vos cartes ressemblent à celles de tout le monde? Es-ce que vous avez des coûts élevées juste pour inclure des cartes sur votre site web?

Passez à OpenStreetMap et découvrez comment vous pouvez créer de magnifiques cartes depuis les meilleures données cartographiques mondiales. Nous vous donnons les données gratuitement; vous pouvez faire toutes les cartes que vous souhaitez avec ces données. Ou vous pouvez bénéficier de l'expertise de ceux qui utilisent déjà OpenStreetMap. Hébergez vos cartes sur votre propre matériel ou ailleurs. Vous avez le contrôle.

switch2osm.org explique comment faire le passage – des premiers principes jusqu'aux tutoriels techniques.
